# Installation Guide

## Prerequisites

- [Bun](https://bun.sh) installed
- [Raycast](https://raycast.com) installed

## Steps

```bash
# Clone repo
git clone https://github.com/snehendu098/rayfine.git

# Enter raycast extension
cd rayfine/raycast

# Install dependencies
bun install

# Run dev mode
bun run dev
```

Raycast will open with the extension loaded in development mode.
